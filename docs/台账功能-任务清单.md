## 阶段二：后端服务层

### 3. 管理对象服务（ledgerService.ts）
- [x] createSubject：创建管理对象 + 自动创建三个分类
- [x] listSubjects：列出所有管理对象（含预算汇总计算）
- [x] updateSubject：更新管理对象（名称、币种、汇率、预算号等）
- [x] deleteSubject：删除管理对象（级联删除）
- [x] getSubjectSummary：计算预算汇总（总预算、已用、剩余、百分比）

### 4. 费用分类服务（ledgerService.ts）
- [x] listCategoriesWithSummary：列出分类（含已用金额、CNY换算）
- [x] getCategoryWithSummary：单个分类汇总（含酒店封顶逻辑）
- [x] updateCategoryBudget：更新分类预算额度

### 5. 费用记录服务（ledgerService.ts）
- [x] createExpense：创建费用记录（含币种）
- [x] listExpenses：列出费用（支持日期/关键词/分类筛选）
- [x] updateExpense：更新费用记录
- [x] deleteExpense：删除费用记录

### 6. 出差单服务（travelOrderService.ts）
- [x] createTravelOrder：创建出差单 + 自动生成补贴
- [x] listTravelOrders：列出出差单（含费用、酒店标准、超额计算）
- [x] updateTravelOrder：更新出差单（自动重算补贴）
- [x] deleteTravelOrder：删除出差单（级联删除费用）
- [x] addTravelExpense：添加差旅费用
- [x] updateTravelExpense：更新差旅费用（含已发/未发状态）
- [x] deleteTravelExpense：删除差旅费用
- [x] buildTravelOrder：构建完整出差单对象（含酒店标准、超额、币种）

### 7. 差旅政策服务（travelPolicyService.ts）
- [x] getPolicy / updatePolicy：获取/更新差旅政策
- [x] getCityTier：境内城市等级查询（4级）
- [x] isPeakSeason：旺季判定（7-10月）
- [x] getHotelStandard：根据城市+日期+目的地类型计算酒店标准
- [x] calculateAllowance：计算补贴（天数、金额、日费率）
- [x] getOverseasCurrency：境外城市→币种映射
- [x] getOverseasHotelTier / getOverseasAllowanceTier：境外等级查询
- [x] resolveCountry：城市→国家映射
- [x] getCityLists：获取境内城市列表（按等级分组）

### 8. AI 分析服务（ledgerAIService.ts）
- [x] analyzeBudget：管理对象整体预算分析（调用 DeepSeek API）
- [x] analyzeCategoryBudget：单分类预算分析（调用 DeepSeek API）

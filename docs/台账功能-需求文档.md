# 台账（Ledger）功能需求文档

## 1. 概述

台账模块是 Calendar 应用的核心财务管理功能，用于管理企业差旅、招待、团建等费用的预算与支出。系统支持多管理对象（如不同项目/部门）、多币种（人民币/欧元/美元）、差旅政策自动化（酒店标准、补贴计算）、AI 预算分析等能力。

## 2. 用户角色

- 普通用户：登录后可管理自己的台账数据，所有数据按用户隔离

## 3. 功能需求

### 3.1 管理对象（Management Subject）

**描述**：管理对象是台账的顶层组织单元，代表一个独立的预算管理实体（如项目、部门）。

**需求**：
- 用户可创建、编辑、删除管理对象
- 每个管理对象包含以下属性：
  - 名称（必填，同一用户下唯一）
  - 币种：人民币（CNY）或欧元（EUR）
  - 汇率：仅 EUR 时有效，表示 1€ = ?¥，默认 7.8
  - 差旅预算号（可选文本）
  - 招待费预算号（可选文本）
  - 成本中心完整代码（可选文本）
  - 横向协同费预算号（可选文本）
- 创建管理对象时自动生成三个费用分类：差旅、招待、团建
- 管理对象卡片展示：总预算、已用金额（原币+CNY）、剩余预算、使用百分比进度条
- 超支时以红色高亮显示
- 删除管理对象时级联删除所有关联数据（分类、费用、出差单）

### 3.2 费用分类（Expense Category）

**描述**：每个管理对象下固定三个分类：差旅、招待、团建。

**需求**：
- 每个分类有独立的预算额度（单位：人民币），可编辑
- 分类卡片展示：额度、已用金额（原币+CNY）、剩余、使用百分比进度条
- 超支时红色高亮
- 支持 AI 分析（按分类维度）

### 3.3 普通费用记录（招待/团建）

**描述**：招待和团建分类下的费用以简单列表形式管理。

**需求**：
- 每条费用记录包含：日期、明细描述、金额、币种（CNY/EUR/USD）
- 支持增删改查
- 支持按日期范围和关键词搜索/筛选
- 列表底部显示合计金额（按币种分组，多币种时显示≈¥换算值）
- 非 CNY 费用在金额旁显示 ≈¥ 换算值

### 3.4 差旅管理

#### 3.4.1 出差单（Travel Order）

**描述**：差旅分类下以出差单为组织单元管理费用。

**需求**：
- 每个出差单包含：标题、目的地类型（境内/境外）、目的地城市、去程日期、返程日期
- 支持增删改
- 境内城市从预设城市列表中选择（分一/二/三/四级城市）
- 境外城市手动输入
- 出差单卡片可展开/折叠，展示总金额（按币种分组）
- 创建出差单时自动根据差旅政策计算并生成补贴记录
- 修改出差单日期/目的地时自动重新计算补贴

#### 3.4.2 差旅费用（Travel Expense）

**描述**：出差单下的具体费用，分为四个子类型。

**需求**：
- 子类型：酒店、机票、其他、补贴
- 每条费用包含：子类型、金额、币种（CNY/EUR/USD）、明细、日期
- 酒店费用额外包含：入住日期、退房日期，自动计算间夜数
- 补贴费用额外包含：已发/未发状态，可切换
- 补贴费用不支持手动选择币种（由系统根据境内/境外自动确定）
- 费用按子类型分组展示，每组显示小计
- 支持增删改

#### 3.4.3 酒店标准与超额

**需求**：
- 根据差旅政策自动确定酒店标准上限（每间夜）
- 境内：根据城市等级和是否旺季确定标准
  - 一级城市（北京/上海/广州/深圳）：区分旺季（7-10月）和非旺季
  - 二级城市：区分旺季和非旺季
  - 三级城市、四级城市：固定标准
- 境外：根据城市所属国家的等级确定标准（7个等级）
- 酒店实际费用超过标准的部分为超额，超额部分不计入预算占用
- 出差单卡片显示酒店标准上限和超标金额
- 单条酒店费用显示超标详情（超额金额、实付/限额对比）
- 所有超额金额统一以人民币（¥）显示

#### 3.4.4 补贴自动计算

**需求**：
- 境内补贴：按天数 × 每日补贴标准（人民币），首尾各半天
- 境外补贴：按天数 × 海外补贴标准（美元或欧元，取决于目的地国家），首尾各半天
- 海外补贴标准分 7 个等级，不同国家/地区对应不同等级
- 欧洲国家使用欧元，其他地区使用美元
- 创建出差单时自动生成补贴记录
- 修改出差单时自动删除旧补贴并重新生成

### 3.5 差旅政策配置

**描述**：用户可自定义差旅政策参数。

**需求**：
- 以弹窗形式展示和编辑
- 可配置参数：
  - 境内每日补贴标准
  - 境内酒店标准（一级城市北京旺季/非旺季、一级城市其他、二级城市旺季/非旺季、三级城市、四级城市）
  - 境外酒店标准（7个等级）
  - 境外补贴标准（7个等级）
- 每个等级旁标注对应的代表性国家/城市
- 所有参数均为数值型，支持实时编辑和保存

### 3.6 汇总视图

#### 3.6.1 差旅子类型汇总

**需求**：
- 差旅 Tab 下提供子 Tab：出差单、酒店、机票、其他、补贴、超额
- 酒店/机票/其他子 Tab：跨出差单汇总该子类型所有费用，支持日期范围筛选
- 显示条目数、合计金额（按币种分组）

#### 3.6.2 补贴汇总

**需求**：
- 独立的补贴汇总视图，支持日期范围筛选
- 分境内/境外分别统计：总额、已发、未发
- 每条补贴可直接切换已发/未发状态
- 表格展示：日期、出差单、明细、金额、状态按钮

#### 3.6.3 酒店超额汇总

**需求**：
- 显示当年酒店超标总额（人民币）
- 逐条列出超标的酒店费用：出差单、目的地、日期、标准、超标金额
- 超标金额详情：实付/限额对比
- 所有金额统一以人民币显示
- 提示"超出部分需年底自付"

### 3.7 AI 预算分析

**描述**：基于 DeepSeek API 的智能预算分析。

**需求**：
- 支持两个维度的分析：
  - 按管理对象整体分析
  - 按单个费用分类分析
- 分析内容包括：各分类占比分析、支出趋势、预算健康度评估、优化建议
- 分析结果以文本形式展示在分类卡片内，可收起
- 分析过程中显示加载动画

### 3.8 Excel 导出

**需求**：
- 支持按管理对象导出 Excel 文件
- Excel 包含三个 Sheet：差旅、招待、团建
- 差旅 Sheet 包含：出差单、目的地、去程、返程、子类型、日期、明细、金额、状态
- 招待/团建 Sheet 包含：日期、明细、金额
- 文件名格式：`{管理对象名称}_台账.xlsx`

## 4. 非功能需求

### 4.1 数据隔离
- 所有台账数据按用户隔离，通过 JWT 认证确保安全

### 4.2 币种处理
- 管理对象级别支持 CNY 和 EUR 两种主币种
- 费用级别支持 CNY、EUR、USD 三种币种
- 所有预算额度和汇总统一以人民币（CNY）计算和展示
- EUR 管理对象下同时显示原币金额和 ≈¥ 换算值
- 酒店超额金额统一以人民币显示

### 4.3 数据完整性
- 删除管理对象时级联删除所有关联数据
- 删除出差单时级联删除所有关联费用
- 金额精度保留两位小数

### 4.4 响应式设计
- 表单和列表支持 flex-wrap，适配不同屏幕宽度
